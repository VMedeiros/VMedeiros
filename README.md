## 🏛️ Architecture & Engineering Principles

I believe the right architecture is the force multiplier for any team. I apply patterns that scale alongside the team and product complexity:

```
┌─────────────────────────────────────────────────────────────┐
│  Presentation    │  Components, Pages, Design System, PWA   │
├─────────────────────────────────────────────────────────────┤
│  Application     │  Commands, Queries, Use Cases, DTOs      │
├─────────────────────────────────────────────────────────────┤
│  Domain          │  Entities, Value Objects, Aggregates,    │
│                  │  Domain Services, Domain Events          │
├─────────────────────────────────────────────────────────────┤
│  Infrastructure  │  Repositories, HTTP, Storage, Cloud      │
└─────────────────────────────────────────────────────────────┘
```

<table>
<tr>
<td width="50%">

### 🎯 What I Practice

- **Clean Architecture** — Strict dependency inversion
- **DDD** — Entities, Value Objects, Aggregates
- **CQRS** — Separated read/write pipelines
- **Reactive Programming** — RxJS, Signals, Observables
- **Design Systems** — Tokens, themes, accessibility
- **Micro Frontends** — Module Federation, Nx monorepos
- **Offline-First** — IndexedDB, Service Workers, PWA

</td>
<td width="50%">

### 📐 Standards I Champion

- Explicit types — zero `any` in production
- 85%+ test coverage (unit, integration, e2e)
- Automated quality gates (pre-commit, CI)
- Code review as a mentoring tool
- Conventional Commits + automated changelog
- Zero magic numbers, debug statements, or dead code
- Living documentation — ADRs + docs as code

</td>
</tr>
</table>

## 🚀 Featured Project

### Krutch — Enterprise CRM Platform

> Enterprise CRM platform built with Angular, Clean Architecture, DDD, and CQRS — from design system to deployment.

<table>
<tr>
<td>

**Highlights:**

- 🏗️ Clean Architecture with strict layered separation
- 🧩 Standalone components with lazy loading and route reuse
- 🎨 Full design token system (light/dark/high-contrast themes)
- 🌐 Complete internationalization (EN, ES, PT-BR)
- 📱 Offline-first PWA (IndexedDB + Service Workers)
- 🔒 JWT auth with guards, interceptors, and refresh token
- 🐳 Dockerized with Nginx for production
- ☁️ Deployed on Firebase + AWS S3/CloudFront
- ✅ Automated quality gates (ESLint + SonarQube + Custom Linters)
- 📊 Interactive dashboards with Chart.js
- 🔄 CI/CD with GitHub Actions + Husky + lint-staged
